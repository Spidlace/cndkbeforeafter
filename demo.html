Default Settings

Default settings

mode: "hover",
showText: true,
beforeText: "BEFORE",
beforeTextPosition: "bottom-left"
afterText: "AFTER",
afterTextPosition: "bottom-right"
seperatorWidth: "4px",
seperatorColor: "#000000",
hoverEffect: true,

Default

With default settings.

$(".beforeafterdefault").cndkbeforeafter();

Mode : drag

With default settings.

$(".beforeafterdrag").cndkbeforeafter( { mode: "drag" } );

Seperator Style

With default settings.

$(".beforeafterdrag").cndkbeforeafter(
{
seperatorColor: "blue",
seperatorWidth: "15px",
}
);

Hide Text

With default settings.

$(".beforeafterdrag").cndkbeforeafter(
{
showText: false
}
);

With default settings.
