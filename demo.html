Default Settings

Default settings

mode: "hover",
showText: true,
beforeText: "BEFORE",
afterText: "AFTER",
seperatorWidth: "5px",
seperatorColor: "#ffffff",
hoverEffect: true,

Default

With default settings.

$(".beforeafterdefault").cndkbeforeafter();

Mode : drag

With default settings.

$(".beforeafterdrag").cndkbeforeafter( { mode: "drag" } );

Seperator Style

With default settings.

$(".beforeafterdrag").cndkbeforeafter(
{
seperatorColor: "blue",
seperatorWidth: "15px",
}
);

Hide Text

With default settings.

$(".beforeafterdrag").cndkbeforeafter(
{
showText: false
}
);
